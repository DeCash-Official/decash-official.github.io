# WalletInfo

:::warning
This API endpoint is not yet implemented
:::

Returns the wallet balance and other information.

## Request Parameters

|     Name    |                Type               | Description |
| ----------- | --------------------------------- | ----------- |
| `address`   | `string`                          | The address of the wallet of which infos will be retrieved. |


## Responses

### Regular response

|     Name    |          Type          | Description |
| ----------- | ---------------------- | ----------- | 
| `delegated` | `boolean`              | Is equal to true if the wallet is a [Delegated Wallet](/docs/concepts/delegated-wallets). |
| `balance`   | `{[k:string]: number}` | Key value pair containing the balance of every held DeCash asset. |

### Validation errors

Returned when the parameters are invalid.

|     Name    |                              Content                              |
| ----------- | ----------------------------------------------------------------- |
| Status code | 400                                                               |
| `errorId`   | `INVALID_PARAMS`                                                  |
| `message`   | `Validation errors in WalletInfo request.`                        | 
| `errors`    | Array of strings containing additional informations.              |
