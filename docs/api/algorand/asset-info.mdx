# AssetInfo

Returns information about every supported token (or about the queried `assetId` token).

## Request Parameters

|     Name    |                Type               | Description |
| ----------- | --------------------------------- | ----------- |
| `assetId`   | `string`                          | The ID of the DeCash asset of which informations will be returned. If not set, informations about all DeCash tokens will be returned. |

## Responses

### Regular response

|     Name    |                           Type                          | Description |
| ----------- | ------------------------------------------------------- | ----------- | 
| `info`      | `Array<{assetId:string, asaId:number, baseFee:number}>` | The array of asset properties. The field `assetId` contains the DeCash Asset ID (e.g. `USDD`); the field `asaId` contains the ID of the actual [algorand asset](../../concepts/tokens#token-addresses) (e.g. 51987792); the field `baseFee` contains the amount that must be paid (of this specific asset) by the sender for each sender-receiver pair. |

### Validation errors

Returned when the parameters are invalid.

|     Name    |                              Content                              |
| ----------- | ----------------------------------------------------------------- |
| Status code | 400                                                               |
| `errorId`   | `INVALID_PARAMS`                                                  |
| `message`   | `Validation errors in AssetInfo request.`                         | 
| `errors`    | Array of strings containing additional informations.              |
