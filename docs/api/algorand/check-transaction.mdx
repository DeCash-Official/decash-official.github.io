# CheckTransaction

Checks the status of a previously submitted transaction. Normally, transactions are mined within 1-5 seconds on the Algorand
blockchain, but it can take a while longer if the blockchain is busy at the moment.


## Request Parameters

|     Name    |                Type               | Description |
| ----------- | --------------------------------- | ----------- |
| `txId`      | `string`                          | The transaction ID received from the [SubmitTransaction](/docs/api/algorand/submit-transaction) method. |


## Responses

This method is not yet implemented in the API.
