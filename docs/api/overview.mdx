# Overview

:::warning
DeCash currently released an **alpha** version of its API for testing. This API may receive breaking changes
at any moment.
:::

DeCash API is the RPC-like API. It uses `POST` methods only and accepts/responds with `application/json`
body content types.


## Using this API

DeCash API is currently in alpha testing phase. You can talk to API using the following endpoints:

+ For test DeCash tokens in the [test Algorand network](/docs/concepts/algorand#networks), use `https://delegator-dev01b.arringo.co/api-test/`
