# Overview

DeCash API is currently a public API anyone can work with.


## Using this API

DeCash API is currently in alpha testing phase. You can talk to API using the following endpoints:

+ For test DeCash tokens in the [test Algorand network](/docs/concepts/algorand#networks), use `https://`
+ For DeCash tokens in the [main Algorand network](/docs/concepts/algorand#networks), use `https://`
