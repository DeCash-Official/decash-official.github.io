# Introduction

DeCash is the independent multi-currency [stablecoin](https://en.wikipedia.org/wiki/Stablecoin) which combines
transparent reserve management with the benefits of the world's most traded fiat currencies. Learn more about DeCash
on [its website](https://decash.com).

This technical documentation is for developers who want to learn about how to integrate with [DeCash API](/docs/api/overview/)
and different use cases for DeCash tokens. Start by reading [DeCash token concepts](/docs/concepts/tokens/).
